Practice Problems

Differential Equations

Basic: First-Order DE

Define differential equations and solve separable first-order equations.

Intermediate: Linear DE

Apply solution methods for linear differential equations.

Advanced: Systems of DE

Solve systems of differential equations using matrix methods.

Stochastic Processes

Basic: Markov Chains

Define Markov chains and compute state transition probabilities.

Intermediate: Stationary Distributions

Find stationary distributions and analyze long-term behavior.

Advanced: Queuing Theory

Apply queuing models to analyze system performance and bottlenecks.

Game Theory

Basic: Normal Form Games

Define game theory concepts and analyze simple strategic games.

Intermediate: Nash Equilibrium

Find Nash equilibria in simultaneous-move games.

Advanced: Mechanism Design

Design incentive-compatible mechanisms for resource allocation.

Boolean Algebra

Basic: Boolean Operations

Define Boolean algebra operations and simplify Boolean expressions.

Intermediate: Karnaugh Maps

Use Karnaugh maps to minimize Boolean functions.

Advanced: Logic Circuits

Design combinational logic circuits from Boolean specifications.

Category Theory

Basic: Categories

Define categories, objects, and morphisms in abstract mathematics.

Intermediate: Functors

Apply functors to map between different mathematical structures.

Advanced: Natural Transformations

Understand natural transformations and their role in category theory.
