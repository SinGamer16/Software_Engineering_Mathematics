Practice Problems

Number Theory

Basic: Modular Arithmetic

Define modular arithmetic and solve linear congruences.

Intermediate: Extended Euclidean

Apply extended Euclidean algorithm to find modular inverses.

Advanced: Chinese Remainder

Apply Chinese Remainder Theorem to solve systems of congruences.

Symmetric Cryptography

Basic: Block Cipher Modes

Define different modes of operation for block ciphers.

Intermediate: AES Implementation

Apply AES encryption algorithm and analyze security properties.

Advanced: Cryptanalysis

Analyze cryptographic attacks and security proofs.

Asymmetric Cryptography

Basic: RSA Algorithm

Define RSA cryptosystem and explain key generation process.

Intermediate: Diffie-Hellman

Apply Diffie-Hellman key exchange protocol for secure key establishment.

Advanced: Elliptic Curves

Analyze elliptic curve cryptography and discrete logarithm problem.

Cryptographic Protocols

Basic: Digital Signatures

Define digital signature schemes and explain their security properties.

Intermediate: TLS Handshake

Apply TLS protocol for secure communication establishment.

Advanced: Zero-Knowledge

Analyze zero-knowledge proof systems and their applications.
